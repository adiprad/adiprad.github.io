Installing development tools for the GBA

Typically, when I am developing for the Game Boy Advance, I use tonclib.

You can install tonc here.

Select all files in the folder, and download them as a zip file.

The file structure should now look like this: 

+-- tonc
|   +-- code
|   +-- bin
|   +-- tonc-text.chm
|   +-- tonc-docs.chm
|   +-- sampleMakefile

The code folder contains source code for tonc examples and tonclib.

The bin folder contains compiled GBA binaries for the examples.

tonc-text.chm and tonc-docs.chm are manuals and documentation, respectively.

sampleMakefile is a standard makefile that I use for building all of my GBA projects.

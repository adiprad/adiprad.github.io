Developing for the GBA: Part 1.5

Before we get into the GBA graphics system, I'll first show you how to set up your project for an IDE, specifically Atom.

If you prefer your own IDE, then you can just set it to run build.bat, but Atom is personally my favorite editor for any project.

First, download Atom at https://atom.io/.

Once you have Atom set up, run apm install build. This will install the open-source build package for Atom (it is a hackable text editor, after all).

Restart Atom, and open your project folder.

Create a file called .atom-build.yml in the root of your project folder.

In .atom-build.yml paste the following: 

cmd: open build.bat

This will tell Atom to open build.bat when you need to build the project. It should work properly on OSX if you installed Wine correctly.

Finally, press CMDALTB on OSX, or CTRLALTB on Windows.

This should run build.bat and successfully build the project.
